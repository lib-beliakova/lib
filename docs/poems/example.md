# Пример для всех!

На прогулке с Кавалером  
Буду я для всех примером!  
Зонтик, шапочка, сапожки --  
Не промокнут мои ножки.  
В отраженьи дивной лужи  
Красоту я обнаружу!

*06.10.2024 г., автору 13 лет*
